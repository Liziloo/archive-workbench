
**Purpose:** Defines the step-by-step path an artifact takes through the app.
*   **Step 1: Ingest.** Scan folder -> Generate Hash -> Check for existing record.
*   **Step 2: Reconcile.** AI proposes a link to Carl’s Catalog or Tropy -> Human confirms.
*   **Step 3: Enrich.** Human transcribes marginalia or corrects dates.
*   **Step 4: Synthesize.** AI drafts a description based on verified facts.
*   **Step 5: Publish.** Record is pushed to Omeka S.
